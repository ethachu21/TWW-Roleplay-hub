# Roleplay Bot 
This right here is a discord bot for the TWW Roleplay discord server!

if you want to use this bot for yourself, you can clone the repository and run it with docker compose :) (although this is not recommended for a production environment)

if you seek to host something like this for yourself, then its recommended to use kubernetes as there is a kubernetes manifest right there!

this is an extremely simple bot and it shouldnt be that hard to use or set up.

if you want to contribute (only if your a member of the TWW RP community) then you may make a pull request and il review it. (ai is allowed just don't make slop)
